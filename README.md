Для клонирования репозитория с GitHub вам нужно выполнить несколько простых шагов. Вот пошаговая инструкция:

### 1. Установите Git
Если у вас еще не установлен Git, сначала установите его:

- **Для Windows**: Скачайте установщик с [официального сайта Git](https://git-scm.com/) и следуйте инструкциям установки.
- **Для macOS**: Установите Git через Homebrew, выполнив команду в терминале:
  ```bash
  brew install git
  ```
- **Для Linux**: Установите Git через пакетный менеджер. Например, для Ubuntu:
  ```bash
  sudo apt-get update
  sudo apt-get install git
  ```

### 2. Откройте терминал или командную строку
- **Windows**: Откройте Git Bash (установленный вместе с Git) или командную строку.
- **macOS/Linux**: Откройте терминал.

### 3. Перейдите в директорию, куда вы хотите клонировать репозиторий
Используйте команду `cd`, чтобы перейти в нужную директорию. Например:
```bash
cd /path/to/your/directory
```

### 4. Клонируйте репозиторий
Выполните команду `git clone`, указав URL репозитория:
```bash
git clone https://github.com/Evools/new-php-blog.git
```

### 5. Дождитесь завершения клонирования
Git начнет клонирование репозитория в текущую директорию. После завершения вы увидите новую пап
